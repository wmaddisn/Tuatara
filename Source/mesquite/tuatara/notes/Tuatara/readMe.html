Installation:

You will find here a directory "modules". Within this directory you will find a directory "tuatara". Please move "tuatara" inside the mesquite folder within your Mesquite_Folder. Thus, when you have moved it, you should have "tuatara", within "mesquite", within "Mesquite_Folder". The next time you start Mesquite, it should load Tuatara (a Tuatara banner will appear in the startup window).
